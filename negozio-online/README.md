# Il Mio Negozio Online 🛍️

Progetto per il corso di Ingegneria del Software - Università degli Studi di Napoli Federico II

## Descrizione
E-commerce sviluppato con tecnologie moderne per la gestione di un negozio online.
Il progetto include un frontend Angular con NgRx per la gestione dello stato e un backend Spring Boot con database MySQL.

## Tecnologie Utilizzate

### Frontend
- Angular 14
- NgRx (Store, Effects, Selectors)
- TypeScript
- CSS3 con layout responsive
- RxJS per la programmazione reattiva

### Backend  
- Spring Boot 2.7
- Spring Data JPA
- Spring Security (configurazione base)
- MySQL 8.0
- Maven per la gestione delle dipendenze
